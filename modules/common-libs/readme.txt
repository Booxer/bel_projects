This folder contains code common to (shared by) some lm32 firmware projects

- b2b (Bunch to Bucket)
- wr-unipz (White Rabbit UNILAC Pulszentrale)
- dm-unipz (gateware between Data Master and UNILAC Pulszentrale)

folders
- include      - contains header files for firmware and x86 code
- fw           - firmware c code
- x86          - x86 code
